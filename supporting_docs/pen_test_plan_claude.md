  Attack Vectors to Test                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                     
  1. File Upload - PDF parsing vulnerabilities, path traversal, malicious file types                                                                                                                                                                 
  2. Search/Query - Injection attacks (NoSQL/SQL depending on your DB)                                                                                                                                                                               
  3. Auth - Broken access control, session handling, rate limiting                                                                                                                                                                                   
  4. API - IDOR, mass assignment, info disclosure                                                                                                                                                                                                    
  5. Infra - TLS config, headers, CORS policy                                                                                                                                                                                                        
                                                                                                                                                                                                                                                     
  Execution Order                                                                                                                                                                                                                                    
                                                                                                                                                                                                                                                     
  1. Recon - Map endpoints with ffuf + ZAP spider                                                                                                                                                                                                    
  2. Automated scans - Nuclei templates, Nikto, ZAP active scan                                                                                                                                                                                      
  3. Manual testing - Burp for file upload abuse, auth bypass, business logic                                                                                                                                                                        
  4. Report - Findings ranked by severity
